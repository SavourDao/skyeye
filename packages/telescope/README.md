## Description

telescope is a dex market price

## Deployment Steps
1. `cd telescope`

2. Copy `.env.example` to `.env`, modify your env config

3. Install the dependencies
```bash
$ npm install
```

4. Running the app use the following command

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
